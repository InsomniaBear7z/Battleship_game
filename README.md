# 🔥🚢 Battleship 2D - Python Game

Một tựa game chiến thuật Battleship được phát triển bằng Python và Pygame. 
Dự án áp dụng các nguyên lý Lập trình hướng đối tượng (OOP) và tích hợp thuật toán AI dò đường cơ bản.

## 🎮🤑 Tính năng nổi bật
- Chế độ AI Dễ (Bắn ngẫu nhiên) và Khó (Thuật toán Truy sát).
- Giao diện đồ họa và hiệu ứng âm thanh đầy đủ.
- Tự động lưu trữ lịch sử trận đấu.

## 🚀 🎶Hướng dẫn cài đặt
1. Yêu cầu hệ thống cài đặt Python 3.12.
2. Cài đặt thư viện: `pip install pygame`
3. Chạy game: `python main.py`
